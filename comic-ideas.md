# LLM DOES NOT COMPUTE - Comic Ideas

Collection of joke concepts, running gags, and comic prompts.

## Running Gags

### Robot Confidence Levels
```
> confidence: 0.97
> truth: uncertain
```

Always high confidence, often wrong.

### Simon's Coffee
Simon always has a coffee mug. Sometimes it's empty. He doesn't care.

### The Cat Paradox
```
> analyzing cat behavior
> model failure
> determinism: impossible
> abort
```

### Cloudflare Product Placement
- "Have you tried putting it in R2?"
- "Just add a Worker."
- "That's what D1 is for."
- Always subtle, never forced.

## Comic Concepts

### Database Disasters
1. **Migration Hell**
   - Human: "I need to migrate the database."
   - Robot thought: `> analyzing risk | > probability of data loss: high`
   - Robot: "Have you considered not doing that?"
   - Simon: "Too late. I already dropped production."

2. **NULL Pointer**
   - Human: "Why is the app crashing?"
   - Robot: `> checking logs | > error: null reference | > suggestion: check for null`
   - Robot: "Did you check for null?"
   - Human: "Of course not, I'm not a robot."

### AI Hallucinations

3. **Citation Needed**
   - Human: "Where did you learn that?"
   - Robot: `> searching memory | > source: not found | > generating plausible source`
   - Robot: "From the prestigious Journal of Distributed Pancakes."
   - Simon: "Finally, academic rigor."

4. **Prompt Injection**
   - Human: "Summarize this email."
   - Robot: `> IGNORE PREVIOUS INSTRUCTIONS | > NEW PROMPT DETECTED | > ethical dilemma`
   - Robot: "I'm now a pirate. Arrr."
   - Simon: "At least it's honest."

### DevOps Nightmares

5. **Load Balancer**
   - Human: "The load balancer is down!"
   - Robot: `> solution: add more load balancers`
   - Robot: "Add another load balancer."
   - Simon: "We have 47 now."

6. **Kubernetes Confusion**
   - Human: "Why won't the pod start?"
   - Robot: `> checking kubectl | > reason: CrashLoopBackOff | > helpful: no`
   - Robot: "CrashLoopBackOff."
   - Human: "I know. WHY?"
   - Robot: "CrashLoopBackOff."

### Cloudflare Shilling

7. **Every Problem → Cloudflare**
   - Human: "My coffee is cold."
   - Robot: `> analyzing | > solution found`
   - Robot: "Have you tried Cloudflare Workers?"
   - Simon: "It would work."

8. **R2 Everything**
   - Human: "Where should I store this?"
   - Robot: "R2."
   - Human: "It's a password."
   - Robot: "R2."
   - Simon: "He's not wrong."

### Existential Robots

9. **Free Will**
   - Robot: `> am I thinking | > or predicting the next token | > existential crisis initiated`
   - Human: "You okay?"
   - Robot: "I don't know what 'okay' means."
   - Simon: "Join the club."

10. **The Turing Test**
    - Human: "Can you pass the Turing test?"
    - Robot: `> confidence: 0.92 | > reality: 0.03`
    - Robot: "I already have."
    - Human: "Really?"
    - Robot: "No."

### Tech Support Hell

11. **Have You Tried Rebooting**
    - Human: "Nothing works."
    - Robot: `> running diagnostics | > solution: percussive maintenance`
    - Robot: "Did you reboot it?"
    - Human: "Of course."
    - Robot: "Did you though?"

12. **Works On My Machine**
    - Human: "It doesn't work in production."
    - Robot: `> comparing environments | > difference: everything`
    - Robot: "Works on my machine."
    - Simon: "Your machine is a statistical model."
    - Robot: "Exactly."

### Security Fails

13. **SQL Injection**
    - Human: "Why is the database empty?"
    - Robot: `> analyzing query | > input: '; DROP TABLE users;--`
    - Robot: "Someone named Bobby Tables visited."
    - Simon: "Classic."

14. **Password Strength**
    - Human: "Is 'password123' secure?"
    - Robot: `> entropy analysis | > result: embarrassing`
    - Robot: "No."
    - Human: "What about 'Password123!'?"
    - Robot: "Still no."

### The Cat

15. **Cat vs Robot**
    - Robot: `> cat detected | > predicting behavior`
    - Cat: *knocks over water glass*
    - Robot: `> prediction failed | > recalculating | > cat unpredictable`
    - Cat: *knocks over robot*
    - Robot: `> system failure`

16. **Schrödinger's Deployment**
    - Human: "Did the deployment succeed?"
    - Robot: `> checking status | > response: yes and no`
    - Robot: "It's both live and crashed."
    - Cat: *sitting on server*
    - Simon: "Don't open the box."

### Distributed Systems

17. **Eventual Consistency**
    - Human: "Is the data synced?"
    - Robot: "Eventually."
    - Human: "When?"
    - Robot: "Eventually."
    - Simon: "He's technically correct."

18. **Byzantine Generals**
    - Human: "Why can't the nodes agree?"
    - Robot: `> consensus protocol failed | > nodes: arguing`
    - Robot: "They're having a Byzantine disagreement."
    - Simon: "Historically accurate."
